# VectorGenEditor 答辩演示步骤

本文档说明答辩展示时的推荐操作流程与界面布局。

## 界面布局说明

主窗口默认尺寸 **1400×850**，采用现代三栏可调整布局：

```
┌─────────────────────────────────────────────────────────────────┐
│ 菜单栏：文件 | 编辑 | 视图 | 生成 | 帮助                          │
├─────────────────────────────────────────────────────────────────┤
│ 顶部工具栏：撤销 重做 | 复制 粘贴 删除 | 保存 导出 SVG              │
├──────┬──────────────────────────────────────────┬─────────────────┤
│ 左侧 │              中间画布工作区               │  右侧选项卡面板  │
│ 绘图 │   （浅灰背景 + 白色画布 + 可选网格）       │  对象属性       │
│ 工具 │                                          │  参数生成       │
│ 栏   │                                          │  图层管理       │
├──────┴──────────────────────────────────────────┴─────────────────┤
│ 状态栏：当前工具 | 鼠标坐标 | 缩放 | 图层数 | 图形数 | 状态        │
└─────────────────────────────────────────────────────────────────┘
```

### 可拖拽分隔条（QSplitter）

- 中央区域为水平 **QSplitter**，包含：左侧工具栏（约 100px）、中间画布（主伸展区）、右侧面板（约 340px）。
- 拖动分隔条可调整画布与右侧面板宽度；左侧工具栏有最小宽度限制，不会被完全挤没。
- 首次启动时布局已预设为答辩友好比例。

### 左侧面板 — 绘图工具

| 按钮 | 功能 |
|------|------|
| 🖱 选择 | 选择与移动对象（默认） |
| ▭ 矩形 | 在画布中心创建矩形 |
| ◯ 椭圆 | 创建椭圆 |
| △ 三角 | 创建正三角形 |
| ⬠ 五边形 | 创建正五边形 |
| ╱ 折线 | 创建折线 |
| T 文本 | 创建文本对象 |

当前选中的工具按钮会高亮显示。

### 右侧面板 — 三个选项卡

#### 1. 对象属性

分组显示选中对象的：

- **基本信息**：类型、ID
- **位置与尺寸**：X、Y、宽度、高度、旋转角
- **外观样式**：边框颜色、填充颜色、线宽、可见性

未选中对象时显示「未选中对象」，所有控件禁用。

属性修改采用 **半即时响应**：数值框在回车或失焦后提交；颜色与可见性勾选后立即生效。所有修改仍走 Command / Undo 栈，可用 Ctrl+Z 撤销。

#### 2. 参数生成

- **星形生成**：根据点数与半径生成可编辑星形
- **箭头生成**：根据起止点生成箭头
- **阵列生成**：按行列与间距生成矩形阵列

#### 3. 图层管理

- 图层列表、新建/删除图层
- 每行可见性复选框
- 显示图层数量与图形总数

### 画布区域

- 外围浅灰工作区，中间白色「纸张」区域带浅色边框
- 菜单 **视图 → 显示网格** 可切换网格显示
- 网格颜色较浅，不抢眼；画布四周留有边距

### 状态栏

实时显示：当前工具、鼠标场景坐标、缩放比例、图层数量、图形数量、操作状态（就绪 / 已选择 N 个对象 / 已保存等）。

### 多选与组拖动

- **Ctrl+A** 或框选（Shift/Ctrl 追加）可选中多个对象；每个选中图形均显示蓝色轮廓
- 多选时属性面板显示「多选 N 个对象」，单对象属性编辑禁用
- 多选后拖动其中任意已选图形，整组一起移动；**Ctrl+Z** 一次撤销整组移动
- 多选时不显示缩放/旋转手柄（避免误导）；单选时才显示 8 个缩放点 + 1 个旋转点

### 工作区边界

- 白色画布区域为合法 **工作区**（默认与 `Document::workspaceRect()` 一致，JSON 可选保存）
- 灰色区域点击创建图形无效；拖动时图形被限制在工作区内
- 参数生成（星形/箭头/阵列）与属性面板修改坐标时，超出工作区的结果会被拒绝或自动夹紧

## 推荐演示流程（约 5–8 分钟）

### 1. 启动与界面介绍（30 秒）

1. 启动 `VectorGenEditor.exe`
2. 简要说明三栏布局、可拖拽分隔条、右侧三个选项卡
3. 指出状态栏信息与顶部/左侧工具栏分工

### 2. 基础绘图（1 分钟）

1. 点击左侧 **▭ 矩形**、**◯ 椭圆**，观察画布中心出现图形
2. 切回 **🖱 选择**，单击选中图形，查看右侧 **对象属性** 自动刷新
3. 拖拽图形移动，说明手柄与选中框对齐

### 3. 属性编辑与 Undo（1 分钟）

1. 选中矩形，在属性面板修改 **X / Y / 宽度 / 高度**，回车或点击其他控件后图形即时更新
2. 修改边框颜色、线宽，观察画布变化
3. 按 **Ctrl+Z** 撤销，再 **Ctrl+Y** 重做

### 4. 参数化生成（1.5 分钟）

1. 切换到 **参数生成** 选项卡
2. **生成星形**：调整点数与半径，点击「生成星形」
3. **生成箭头**：设置参数后「生成箭头」
4. **生成阵列**：设置行列与间距后「生成阵列」

### 5. 图层与文件 I/O（1.5 分钟）

1. 打开 **图层管理**，新建图层，切换可见性
2. **文件 → 保存 JSON**，保存当前项目
3. **文件 → 新建** 或关闭后 **打开 JSON**，验证图形与属性恢复
4. **文件 → 导出 SVG**，用浏览器或 Inkscape 打开验证

### 6. 多选拖动与工作区（1 分钟）

1. 创建 5 个以上图形，**Ctrl+A** 全选，确认每个图形均有选中轮廓，状态栏显示数量
2. 拖动其中一个，整组一起移动；**Ctrl+Z** 一次恢复
3. 在灰色区域尝试创建文本/直线，应无新图形
4. 将图形拖向白色画布边缘，确认被限制在边界内

### 7. 视图与缩放旋转（1 分钟）

1. **视图 → 放大 / 缩小 / 适应窗口**
2. 切换 **显示网格**
3. 拖动 QSplitter 分隔条，展示布局可调
4. 单选图形，演示 8 个缩放手柄与旋转手柄；属性面板修改宽高/角度后画布同步

### 8. 关于（可选）

**帮助 → 关于** 展示技术栈与架构说明。

## 答辩话术要点

- **定位**：参数化矢量图形生成与编辑系统，非简单 Qt Demo
- **架构**：Model-View-Command，Document 为唯一数据源，所有写操作经 QUndoCommand
- **坐标系**：局部顶点 + Transform 统一，选中框与手柄精确对齐
- **UI**：QSplitter 可调整布局、中文面板、即时属性反馈，适合实际编辑工作流

## 人工验收 Checklist

演示前请逐项确认：

- [ ] 窗口默认约 1400×850，三栏布局清晰
- [ ] 可拖动分隔条调整画布与右侧面板宽度
- [ ] 左侧工具互斥高亮，当前工具在状态栏显示
- [ ] 顶部工具栏撤销/重做/复制/粘贴/删除/保存/导出均有效
- [ ] 菜单项无「假按钮」；未实现功能已隐藏或标注
- [ ] 未选中时属性面板显示「未选中对象」且控件禁用
- [ ] 选中后属性自动加载；改 X/Y/宽高/旋转后图形更新
- [ ] 颜色、可见性修改立即生效
- [ ] Ctrl+Z / Ctrl+Y 可撤销/重做属性修改
- [ ] 参数生成三个按钮均可生成图形
- [ ] 图层新建/删除/可见性正常，数量统计正确
- [ ] 网格可显示/隐藏，画布白底+浅灰工作区
- [ ] 状态栏坐标、缩放、数量、状态文字正确更新
- [ ] JSON 保存/打开后属性与位置正确
- [ ] Ctrl+A 全选后所有图形有选中轮廓；多选时属性面板显示「多选 N 个对象」
- [ ] 多选拖动整组移动；Ctrl+Z 一次撤销整组移动
- [ ] 灰色区域不能创建图形；拖动/缩放不能超出白色工作区
- [ ] 单选时缩放手柄与旋转手柄可用；Ctrl+Z 可撤销缩放/旋转
